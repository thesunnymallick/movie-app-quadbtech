## How to Use


- Clone the repository:
```bash
git clone https://github.com/thesunnymallick/movie-app-quadbtech
```
- Install the dependencies:
```bash
npm install
```

- Start the app:

```bash
npm run dev
```






## Tools/Libraries
- React.js
- SCSS
- Axios
